## Fast and Slow pointers

快慢指標多用於 Linked-List 的問題，透過 slow 與 fast 兩根指標，讓兩個指標保持一定的間隔規律，通常可以用於解決以下幾種問題：

- 找到 Linked-List 中心點
- 找到 Linked-List 倒數第 K 個節點
- 判斷一個 Linked-List 有沒有循環
